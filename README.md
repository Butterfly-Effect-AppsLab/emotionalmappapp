# Startup project

## Backend init
    cd backend
    python -m venv venv
    Linux source venv/bin/activate / Windows venv\Scripts\activate
    pip install -r requirements.txt
## Frontend init
    cd emuapp
    npm install

## Start Project
    venv\Scripts\activate
    cd emuapp
